🎮 MENU OPTIONS EXPLAINED:

Option 1: Setup/Initialize Database
   → Creates 5,000 nodes and 25,000+ edges
   → Takes 30-60 seconds first time
   → Creates file: graph_data.db
   → RUN THIS FIRST!

Option 2: Find Shortest Path
   → Enter two node IDs (1-5000)
   → Choose Sequential or Parallel algorithm
   → See results with path and distance
   → Example: Find path from Node 100 to Node 2500

Option 3: Run Performance Analysis
   → Tests both sequential and parallel
   → Creates: performance_report.txt
   → Creates: performance_data.csv
   → Shows speedup metrics
   → Takes 2-5 minutes

Option 4: View Database Statistics
   → Shows total nodes, edges, paths
   → Shows connectivity metrics

Option 5: Exit
   → Close the application

❓ TROUBLESHOOTING:

Problem: "Python not found"
Solution: Install Python from python.org
          Check "Add Python to PATH"

Problem: File doesn't open with double-click
Solution: Right-click the file
          Select "Open with" → Command Prompt (Windows)
          Or open Terminal and drag the file into it

Problem: Takes forever on first run
Solution: Normal! Database generation takes 30-60 seconds
          Be patient, don't close the window

Problem: "Permission denied" on Mac/Linux
Solution: Right-click RUN_ME_FIRST.sh
          Select "Get Info"
          Check "Execute"
          OR in Terminal: chmod +x RUN_ME_FIRST.sh
